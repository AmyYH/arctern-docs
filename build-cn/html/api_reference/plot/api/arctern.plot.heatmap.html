arctern.plot.heatmap¶

arctern.plot.heatmap(ax, points, weights, bounding_box, map_zoom_level=None, coordinate_system='EPSG:3857', aggregation_type='max', **extra_contextily_params)[源代码]¶

在 Matplotlib 中绘制热地图。

参数

  • ax (matplotlib.axes.Axes) – 绘制几何图形的画布的坐标轴。

  • points (GeoSeries) – 一组点。

  • weights (Series) – 点密度的权重。

  • bounding_box (list) – 地图的边界，如 [west, south, east, north]。

  • map_zoom_level ([type], optional) – 地图的缩放级别，默认值为 ‘auto’。

  • coordinate_system (str, optional) – 为所有几何图形设置 CRS（Coordinate Reference System，坐标参考系），默认值为 ‘EPSG:3857’。目前仅支持 SRID（Spatial Reference Identifier，空间参考标识符）作为 CRS 的 WKT 表示。

  • aggregation_type (str, optional) – 聚合类型，默认值为 ‘max’。

  • **extra_contextily_params (dict) – 传到 contextily.add_basemap 中的其他参数。
